### Описание компонентов
__Tablet (Web POS)__ – обычный браузер на планшете. Отображает интерфейс, общается с облаком через HTTPS/WebSocket. Не содержит драйверов и не хранит состояние.

__Edge Gateway (HAL)__ – легковесный сервис (Python/Go) на локальном устройстве (можно использовать старый системный блок без лицензий, или Raspberry Pi 4). Отвечает за:

Подключение к фискальному регистратору через USB/COM (драйверы для разных моделей – инкапсулированы).

Подключение к банковскому терминалу через USB/Ethernet (использует универсальные протоколы: ARCUS, OpenPay, или через локального посредника).

Предоставление REST/WebSocket API для облачного POS.

Буферизацию транзакций при потере связи (офлайн-режим).

Cloud POS Backend – микросервисы в облаке (например, AWS ECS). Управляет каталогом, ценами, промоакциями, синхронизацией.

Managed DB (RDS) – центральное хранилище справочников, остатков, лояльности.

Payment Aggregator (Cloud) – внешний сервис для безналичных платежей (можно использовать тот же шлюз, что и Edge, либо облачное решение типа Stripe).
